# Installation

1. git clone https://github.com/yinfredyue/djreact.git
2. cd backend  
    python -m venv env  
    source env/bin/activate   
    pip install django djangorestframework django-cors-headers    
    cd backtrack  
    python manage.py runserver  

3. Go to another terminal
    cd frontend/ui
    yarn install
    yarn add antd --save
    yarn add axios --save
    yarn start
